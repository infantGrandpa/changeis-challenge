# Changeis Coding Challenge

A React Application that fetches data from the [FakerAPI](https://dog.ceo/api/breeds/image/random) and
displays it in a simple page.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI

## Getting Started
1. Install [Node.js](https://nodejs.org/en/download) and npm
2. Clone the repository or download as a zip file.
3. In the root folder, install dependencies: `npm install`
4. Run the development server: `npm run dev`
